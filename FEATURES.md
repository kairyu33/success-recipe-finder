# 機能一覧

## コア機能

### 1. AI駆動ハッシュタグ生成

- **Claude 3.5 Sonnet統合**: 最新のAIモデルを使用
- **コンテキスト理解**: 記事の主題、テーマ、キーワードを深く分析
- **20個のハッシュタグ**: note.comに最適な数を自動生成
- **日本語最適化**: 日本語コンテンツに特化したプロンプト設計

### 2. 使いやすいUI/UX

- **大規模テキスト対応**: 10,000文字まで入力可能
- **リアルタイム文字カウント**: 現在の入力文字数を表示
- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **アクセシブル**: キーボードナビゲーション対応

### 3. コピー機能

- **ワンクリックコピー**: 各ハッシュタグを個別にコピー
- **一括コピー**: すべてのハッシュタグをスペース区切りでコピー
- **視覚的フィードバック**: コピー完了時にチェックマークアイコン表示
- **自動リセット**: 2秒後に自動的に元の状態に戻る

### 4. エラーハンドリング

- **入力バリデーション**: 空のテキスト、文字数制限をチェック
- **API エラー処理**: Anthropic APIのエラーを適切にハンドル
- **ユーザーフレンドリーなメッセージ**: わかりやすいエラー表示
- **リトライ対応**: エラー後も再試行可能

### 5. ローディング状態

- **アニメーション**: スピナーアイコンで処理中を表示
- **ボタン無効化**: 処理中は再クリックを防止
- **進捗メッセージ**: 「ハッシュタグを生成中...」を表示

## UI/UXの詳細

### デザイン

- **グラデーション背景**: 美しいブルー/インディゴグラデーション
- **カードレイアウト**: 白い背景に影付きで見やすい

## 技術的特徴

### コード品質

- **ESLint**: Next.js推奨設定
- **Prettier対応**: フォーマット可能
- **コメント**: 主要な関数にJSDoc形式のドキュメント

## API設計

### エンドポイント: POST /api/generate-hashtags

**リクエスト:**
```typescript
{
  articleText: string;  // 1-10,000文字
}
```

**レスポンス (成功):**
```typescript
{
  hashtags: string[];   // 20個のハッシュタグ配列
}
```

**レスポンス (エラー):**
```typescript
{
  error: string;        // エラーメッセージ
}
```

### バリデーション

1. APIキーの存在確認
2. 記事テキストの型チェック
3. 文字数制限（最大10,000文字）
4. 空文字チェック

### エラーハンドリング

- **400 Bad Request**: 無効な入力
- **500 Internal Server Error**: API設定エラー、Claude APIエラー
- **カスタムエラーメッセージ**: 問題に応じた具体的なメッセージ

## 将来の拡張可能性

### 予定している機能

1. **ハッシュタグ履歴**: ローカルストレージで過去の生成結果を保存
2. **お気に入り**: 気に入ったハッシュタグをお気に入りに追加
3. **カテゴリ選択**: 記事のカテゴリを選んで最適化

6. **複数言語対応**: 英語、韓国語などのハッシュタグ生成
7. **トレンド分析**: note.comのトレンドを反映
8. **A/Bテスト**: 複数のハッシュタグセットを生成して比較

### カスタマイズポイント

1. **プロンプトのカスタマイズ**: より具体的な指示を追加
2. **ハッシュタグ数の変更**: 20個以外にも対応
3. **UIテーマ**: カスタムカラースキーム
4. **言語設定**: 多言語対応
5. **AIモデル切り替え**: Claude以外のモデルにも対応可能
